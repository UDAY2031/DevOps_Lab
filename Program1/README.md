# ✅ User Registration Form Project — Git Setup & GitHub Push

## 1️⃣ Create Project Folder
```
mkdir event-registration
cd event-registration
```

---

## 2️⃣ Create an HTML File
Create a file named **index.html** and add a simple user registration form  
(with fields: **Name, Email, Phone**).  
*(Write any simple HTML — even 5–6 lines is enough.)*

---

## 3️⃣ Initialize Git
```
git init
```
## 4️⃣ Add & Commit Files
```
git add index.html
git commit -m "Add simple event registration form"
```

---

## 5️⃣ Create GitHub Repository  
Go to GitHub → New Repository → Name it **event-registration**.

---

## 6️⃣ Connect Local Repo to GitHub
```
git remote add origin https://github.com/YOUR_USERNAME/event-registration.git
```

---

## 7️⃣ Push to GitHub

```
git branch -M main
git push -u origin main
```

---

## 8️⃣ Verify Git Setup
Check commit history:
```
git log --oneline
```

Check remote:
```
git remote -v
```

---

✔ Done! Your project is now pushed to GitHub.  
You only created **one HTML file + Git commands**, simple and clean.
